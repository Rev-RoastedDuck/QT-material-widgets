## 基本功能
- ✔ 提供14种**美化按钮**。
- ✔ 提供3种**美化输入框**。
- ✔ 提供5种**炫酷的组件容器**。
- ✔ 提供1种支持**3种状态的通知组件**。
- ✔ 提供2种**美观滑块组件**。
- ✔ 提供4种**仪表盘组件**。
- ✔ 提供美观的**复选框和下拉框（组合框）**。

## 快速开始
1. 将 `rrd_widget` 放置于 Python 的 `Lib` 目录下：
   ```
   cp -r ./python/rrd_widgets $(python -c "import sys; print(sys.prefix + '/Lib')")
   ```
2. 进入 `examples` 目录，在那里可以找到各种组件的示例。
