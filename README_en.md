## Quick start
1. Place `rrd_widget` in the Python `Lib` directory.

   ```
   cp -r ./python/rrd_widgets $(python -c "import sys; print(sys.prefix + '/Lib')")
   ```
2. Enter the `examples` directory.There, you can find examples of various components.
